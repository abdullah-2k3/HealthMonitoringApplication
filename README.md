# Healthline Pro | Web Application

This project is a health monitoring web application built using Python Flask, Javascript, and SQL Server database. The application is designed to allow users, doctors, and admins to sign up and have their personalized dashboards and modules.

## Features

- User, doctor, and admin sign-up and login functionality
- Personalized dashboards for each user type
- Modules for tracking and monitoring various health indicators
- Appointment bookings
- Health resources and expert opinions
- Record of patient health history
- FAQs
- SQL Server database for storing user data

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository to your local machine
2. Install the required packages and dependencies
3. Set up the SQL Server database
4. Run the application using the Flask development server

## Usage

Once the application is up and running, users can sign up and log in to access their personalized dashboards. From there, they can access various modules for tracking and monitoring their health indicators.

Doctors and admins have additional functionality, such as the ability to view and manage user data.
